# UserApi_Dmoney

## Technology and Tools used 
  - PostMan 
  - Newman 
  - VSCode 
  
## Scenario
  - Login 
  - Create Account 
  - Deposit to Agent from System 
  - Deposit to Customer1 from Agent 
  - Send money 
  - Statement & Balance check
  
## How to run this project 
  - Clone the project 
  - Hit the following command 
  ``` npm i ```
  ``` npm test ```
  
## Prerequisite 
  - Node.js must be installed
